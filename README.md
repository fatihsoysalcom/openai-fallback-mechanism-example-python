# OpenAI Fallback Mechanism Example Python

This example demonstrates a fallback mechanism for interacting with an external API, such as OpenAI. It simulates a primary API call that can fail (e.g., due to billing issues or network problems) and then gracefully switches to a simpler, local fallback function to ensure system resilience and continuity. Two scenarios are shown: one where the primary API fails and fallback is used, and another where the primary API succeeds.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
